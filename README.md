# IU Admission Frontend

Academic project for `Software Systems Design` course. 

Should be used with backend:
https://github.com/ssd-team/IUAdmissionSystem

## Quick start

Quick start options:

- `npm install`
- `npm start`

For optimized variant:
- `npm run build`
- `npm start build`

To stop:
- `npm stop` (stops all npm servers)
